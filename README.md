# Screen Pulse Remote Support

This repository contains the backend and web portal for the Screen Pulse Remote Support system.

## Structure

*   `remote-backend/`: Node.js backend (REST + WebSocket)
*   `remote-web/`: React/Vite web portal

## Getting Started

### Backend
1.  Navigate to `remote-backend/`.
2.  Run `npm install`.
3.  Run `npm run dev` to start the server on port 8080.

### Web Portal
1.  Navigate to `remote-web/`.
2.  Run `npm install`.
3.  Run `npm run dev` to start the web app on port 3000.
